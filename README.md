# m17n_ibus_persian

## Introduction

m17n Ibus entry methods for Persian (Farsi) text entry

## Installation

To install, move the fa-banan.mim file to ~/.m17n.d and restart Ibus. The input method should show up in Preferences->Input Method->Add->Persian->fa-banan.
